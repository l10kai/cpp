# 刷题心得体会



 ### 二分

> 二分是一开始就学习的基础算法之一，是最常用的算法之一，其基本模板有两种：、
>
> ```cpp
> //模板一
> int l =0 ,r = n;
> while(l < r){
>     int mid = l + r >> 1;
>     if(check()) r =mid;
>     else l = mid + 1;
> }
> //模板二
> int l = 0 , r = n;
> while(l < r){
>     int mid = l + r +1>> 1;
>     if(check()) l = mid;
>     else r = mid - 1;
> }
> ```
>
> 模板非常好记，但是近期在具体题目中经常出现对二分模板混淆使用导致的一系列不应该的错误。
>
> 一句话，**模板1找答案时会尽量往左边找，就会找到该数的起始位置；而模板2找答案时会尽往右边找，就会找到该数的终止位置。**
